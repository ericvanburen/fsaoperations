Date of Birth Changes

Documents Necessary:

  • Copy of Drivers License
  • Copy Passport
  • State or Federal issued identification card
  • Copy of Birth Certificate, or
  • Other official documentation

Action Required:

  1. Open F110 Screen
  2. Type Social Security Number (SSN),
  3. “Hit “ Enter Key
  4. Type Date Of Birth (MM/DD/YYYY) in Changed value Screen,
  5. “Hit” Enter Key
  6. Notate L102 screen

Name Changes

Documents Necessary

  1. Identification
  • Copy of Drivers License,
  • Passport,
  • State or Federal issued identification card, or
  • Other official documentation

2. Social Security Card

Action Required

  1. Open L103 Screen
  2. Type SSN
  3. “Hit “ Enter Key
  4. Type new name,
  5. “Hit” Enter Key (old name will appear in “PREV NAME” area).
  6. Notate L102 screen

Federal Offset Removal

Reasons:

  • Bankruptcy
  • Account paid in full
  • Account settled in full
  • Other

Actions

  1. Open I100 Screen
  2. Type SSN
  3. “Hit “ Enter Key
  4. “Hit” F8 Key to scroll Screens looking for “Certified” debts (Accounts are certified at the debt level).
  5. Type “I”, (to inactivate account)
  6. “Hit” Enter Key to Inactivate Account
  7. Notate L102 screen
